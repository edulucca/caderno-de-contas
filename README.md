# caderno-de-contas
## Caderno de contas é um software que te permite registrar suas contas mensais.
### Projeto referente a matéria Integração Contínua e DevOps da Pós-Graduação MIT em Engenharia de Software com Java do Instituto Infnet.

<br/>

> Tecnologias utilizadas
- Java 19
- Spring Boot
- Spring Data JPA
- Lombok
- Junit 5
- Spring Boot Test
- Spring Boot Actuator
- Terraform
- Docker
  * MySQL
  * API
- Prometheus
- Micrometer
- Zipkin
- Papertrail

> Instruções de execução do programa
* No terminal subir o docker compose com ``docker-compose up``.
* No diretórios ``REQUESTS > DIVIDA`` têm as requisições para serem usadas em ferramentas como o Postman etc.
